## 2026-07-23 - GitHub Lacks Native DMs

**Vulnerability:** Contact instructions in documentation (like SECURITY.md or CODE_OF_CONDUCT.md) directed users to send direct messages on GitHub. Since GitHub does not support private direct messaging, users might attempt to report sensitive information (like security vulnerabilities or code of conduct violations) publicly.
**Learning:** GitHub lacks a direct messaging feature. Instructions should explicitly direct users to private channels (like email) to prevent sensitive information from being publicly disclosed.
**Prevention:** When providing contact instructions in documentation, direct users to the email listed on the GitHub profile rather than instructing them to send a direct message on GitHub.
