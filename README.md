## Apps

| App | What it does | Stack | Latest |
|-----|--------------|-------|--------|
| <img src="assets/4charm.png" width="20" height="20" alt=""> [**4Charm**](https://github.com/RazorBackRoar/4Charm) | High-performance 4chan media downloader with smart organization and resume | Python · PySide6 | [v2.0.0](https://github.com/RazorBackRoar/4Charm/releases) |
